# 🎓 Student Management App

A comprehensive and user-friendly application designed to streamline student data management. This app allows you to easily add new students, maintain accurate student counts, and manage overall capacity or quantities efficiently.

## ✨ Features

* **Add Students:** Quickly register new students with their essential details.
* **Maintain Count:** Automatically track and display the total number of enrolled students.
* **Manage Quantity/Capacity:** Monitor and control classroom or course capacities to ensure optimal student distribution.
* **Intuitive Dashboard:** A clean interface for managing all student-related activities.
